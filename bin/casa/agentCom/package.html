The package (agentCom = "Agent Communications") handles connections between agents.  The class Channel is an abstraction of a communication channel. Objects of subclasses of Channel are embedded in URLDescriptors.  DirectChannel (<:Channel) is a "channel" to a local agent in the same process.  TCPChannel (<:Channel) is TCP/IP socket channel, which works with SocketChannelGlobal to do two-way remote communication.
